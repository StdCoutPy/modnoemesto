# 👗 Modnoe Mesto — AI-Powered Fashion E-commerce

Современный интернет-магазин одежды на Django, интегрированный с искусственным интеллектом для персонализации стиля и автоматизацией наполнения контента. Проект сочетает в себе классический e-commerce и Fashion-Tech решения.

## 🌟 Ключевые особенности

### 🤖 AI & Automation (Уникальные фишки)
*   **AI-Стилист:** Встроенный чат-помощник на базе OpenAI/Google Gemini, который консультирует пользователей по подбору образов и трендам.
*   **Smart Lookbook Parsing:** Кастомные скрипты для автоматического сбора трендовых образов с Pinterest для формирования визуальных подборок.
*   **Auto-DB Seeding:** Скрипты автоматического наполнения базы данных товарами и лукбуками для быстрого развертывания витрины.

### 🛒 E-commerce логика
*   **Управление складом:** Система учета остатков по размерам (`ProductSize`). Автоматическое списание товара из БД при успешной оплате.
*   **Гибкая система скидок:** Генерация персональных промокодов при регистрации и динамический пересчет корзины (AJAX).
*   **Интеграция оплат:** Имитация платежного шлюза Freedom Pay (Paybox) с логикой проверки цифровой подписи (PG_SIG).
*   **Telegram Auth:** Быстрая авторизация и верификация пользователей через Telegram Bot API.

## 🛠 Технологический стек

*   **Backend:** Python 3.10+, Django 4.2+
*   **Database:** SQLite (Dev) / PostgreSQL (Ready)
*   **Async & AI:** OpenAI API, Google Generative AI, `python-telegram-bot`
*   **Frontend:** Tailwind CSS, JavaScript (AJAX)
*   **Deployment:** Docker, Gunicorn, Nginx, PythonAnywhere/VPS
